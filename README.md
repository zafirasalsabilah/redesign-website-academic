The website redesign project was carried out for the Mechanical Engineering, Law, and Master of Law study programs at Universitas Pamulang. The main focus was improving the user interface, navigation, and the presentation of academic, research, and community service information to make the websites more responsive, well-structured, and easier for users to access.
